# Microservicio Usuarios
Este microservicio es uno de los más importantes ya que de el estan anidadas muchas funciones por ejemplo añadir a X usuario con rol de jugador a un equipo o permitir hacer cambios si su rol es de Arbitro o Admin.
Tiene conexión con equipos y auditoria ya que al hacer cambios como guardar, crear o actualizar se guardara en la base de datos para tener un historial de dichos cambios.

## Dependencias
* Spring Web
* Validation
* Spring Data JPA
* OpenFeign
* MySQL Driver
* Lombok
